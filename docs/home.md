## 接口地址

| host     | 192.168.XXX.XXXX:XXXX |
| -------- | :-------------------: |
| basePath |        course         |

## 请求状态 code

| code （int 类型） |       状态码描述       |
| :---------------: | :--------------------: |
|        200        |          正常          |
|        401        |         未登录         |
|        402        |         未注册         |
|      其他值       | 异常(msg 返回错误原因) |

## 分页

?> 如果返回的是数据是 list,并且需要分页的时候。

|        参数        |   描述   |
| :----------------: | :------: |
| page （int 类型）  | 当前页数 |
| total （int 类型） |  总数量  |

## 返回格式

```js
data:{
  code:1,
  msg:'正常',
  page:1,
  total:99,
  data:{}
}

```
